Document Extraction Web App — Design Brief
Product Vision
A bilingual (EN/TH) document extraction platform enabling users to extract structured data from documents using pre-built templates or a general extraction option, with a freemium pricing model.
________________________________________
Design Reference
Inspiration for navigation : LlamaCloud / LlamaExtract (https://www.llamaindex.ai/llamaextract) 
https://cloud.llamaindex.ai/
________________________________________
Navigation Structure
Public (Unauthenticated)
•	Home
•	Pricing
•	About
•	Sign In
Authenticated
•	Dashboard (home)
•	General Extraction (for any document type)
•	Templates (4): Bank Statements, Invoices, Purchase Orders, Contracts
•	History
•	Settings
________________________________________
Core Features
Authentication
•	Social login options: LINE, Google, Facebook, Apple
•	LINE prioritized for Thai market
Extraction Options
General Extraction
•	For any document type without a predefined template
•	User-defined or auto-detected fields
Templates (4 Pre-built)
•	Bank Statements
•	Invoices
•	Purchase Orders
•	Contracts
Each template should be visually distinct for quick recognition.
Extraction Workflow
•	Two-panel layout: Upload area (left) | Results with confidence scores (right)
•	Drag-and-drop file upload
•	Real-time extraction results
•	Inline editing for corrections
•	Export options: JSON, CSV, Excel
History
•	View past extractions
•	Search and filter capability
•	Access to re-download results
Freemium Model
•	Free tier: 100 pages/month
•	Pro tier: 1,000 pages/month
•	Usage indicators visible throughout
•	Warnings when approaching limit
•	Upgrade prompts when limit reached
________________________________________
Bilingual Support
Locales
•	English (en-US)
•	Thai (th-TH)
Requirements
•	Language switcher accessible throughout
•	Thai typography support (recommend: Noto Sans Thai)
•	Thai date format: Buddhist calendar year (พ.ศ.)
•	All UI labels, messages, and field names in both languages
________________________________________
Summary
Deliver a clean, intuitive interface that allows users to extract data using general extraction or 4 pre-built templates, upload files, view results with confidence scores, and manage usage within a freemium model—all fully supported in English and Thai.
